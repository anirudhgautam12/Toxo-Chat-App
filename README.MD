🚀 TOXO – Real-Time Chat App

TOXO is a modern real-time chat application that enables seamless communication with instant messaging, authentication, and user presence tracking. Built with a scalable full-stack architecture and optimized for performance.

✨ Features
🔐 JWT Authentication
💬 Real-time messaging using Socket.IO
🟢 Online/Offline user status
👤 Profile management
🖼️ Image upload via Cloudinary
⚡ Fast and responsive UI
🛠️ Tech Stack

Frontend:

React + Vite
Zustand (State Management)
Tailwind CSS

Backend:

Node.js
Express.js

Database:

MongoDB

Real-Time:

Socket.IO
📂 Project Structure

TOXO/
│── frontend/ # Frontend (React + Vite)
│── backend/ # Backend (Node.js + Express)
│── README.md

⚙️ Setup Instructions
🔧 Backend Setup

cd backend
npm install
npm run dev

💻 Frontend Setup

cd frontend
npm install
npm run dev
